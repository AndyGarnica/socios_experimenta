La tipografía web y para usos digitales de Experimenta es MULI,
Se pueden usar las variantes de la fuente (en web usar preferentemente
un máximo de 5 diferentes pesos de fuente, idealmente sólo 3 pesos de la fuente
-nunca extralight, ni extrabold o black- )


https://fonts.google.com/?query=muli&selection.family=Muli


Recomendado para HTML:
<link href="https://fonts.googleapis.com/css?family=Muli:300,400,400i,600,700" rel="stylesheet">


CSS:
font-family: 'Muli', sans-serif;



@import:

<style>
@import url('https://fonts.googleapis.com/css?family=Muli:300,400,400i,600,700');
</style>
